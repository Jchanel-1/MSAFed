## Data preparation
* Fed-polyp: Download the corresponding datasets and follow by [here](https://github.com/DengPingFan/PraNet)
* Fed-prostate: Download the corresponding datasets [here](https://liuquande.github.io/SAML/)
* Fed-isic : Download the corresponding datasets [here](https://challenge.isic-archive.com/data/).

After downloading the datasets, the file structure should be:

├── Dataset
  ├── FedPolyp_npy
    ├── client1
      ├── sample1.npy, sample2.npy, ...
    ├── client2
    ├── ...
